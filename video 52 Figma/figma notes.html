Core Concepts

Learn the fundamentals that every Figma designer should know

Frames vs Groups

  • Frames define canvas areas (like artboards)
  • Groups just bundle elements together
  • Use frames for:
    • Responsive constraints
    • Auto-layout containers
    • Prototyping
Shortcut: F for Frame tool

Auto Layout

  • Vertical or horizontal stacking
  • Set spacing between items
  • Padding controls internal space
  • Resizing options:
    • Fixed (default)
    • Hug contents
    • Fill container
Shortcut: Shift + A

Components

  • Create with ⌘ + ⌥ + K
  • Main Component = "Source of truth"
  • Instances inherit changes
  • Override text, colors, visibility
  • Use Variants for different states

Constraints

  • Controls how elements resize in frames
  • Set horizontal/vertical behavior
  • Options: Left, Right, Center, Scale
  • Works with Auto Layout
Protip: Use for responsive designs

Prototyping

  • Connect frames with interactions
  • Trigger types: On Click, While Hovering, etc.
  • Animation options:
    • Smart Animate
    • Easing curves
    • Timing controls

Design Systems

  • Color Styles (Text, Fill, Stroke)
  • Text Styles (Type hierarchy)
  • Component Library
  • Publish to Team Library
Shortcut: ⌘ + / to browse assets

Keyboard Shortcuts

Work faster with these essential Figma shortcuts

Navigation

  • Space + Drag = Pan
  • Z + Click = Zoom In
  • ⌘ + 0 = Zoom to Fit
  • ⌘ + G = Toggle Grid

Editing

  • ⌘ + D = Duplicate
  • ⌘ + E = Export
  • ⌘ + R = Rename Layer
  • ⌘ + ⌥ + C = Copy as CSS

Text

  • T = Text Tool
  • ⌘ + B = Bold
  • ⌘ + ⌥ + L = List
  • ⌘ + ⌥ + H = Horizontal Align
