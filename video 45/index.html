CSS Transition Property

What is CSS Transition?

CSS Transition allows you to make changes in CSS properties smoothly over time, instead of instantly.

transition: property duration timing-function delay;

BreakDown

Part Explanation Example
property Which CSS Property to animate (like transform, bg-color) transform or all
duration How long the animation takes? 0.3s, 2s, 500ms
timing-function Speed Curve (ease, linear, ease-in, ease-out, cubic-bezier) ease or linear
delay Time to wait before starting the animation. 0s, 2s, 0.5s, 200ms
