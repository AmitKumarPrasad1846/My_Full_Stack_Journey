box-shadow CSS property

box-shadow creates a shadow outside an element.

Syntax:

box-shadow: offset-x offset-y blur-radius spread-radius color;

Breakdown
Part Function Example value Effect
offset-x It moves shadow horizontally. 5px Moves right
offset-y It moves shadow vertically. 10px Moves down
blur-radius It blur's the shadow. 15px More blur = Soft shadow
spread-radius It makes the shadow either big or small. 0px Positive = Big shadow, Negative = Small shadow
color Color of the shadow. rgba(0, 0, 0, 0.5) Black semi-transparent

Simple shadow

box-shadow: 5px 5px 10px gray;

  • 5px right, 5px down, blurred by 10px, color: gray

Inset shadow

box-shadow: inset 5px 5px 10px black;

  • "inset" makes the shadow to go inside the box.[jaise box ke andar depth ho]

Multiple shadows on a single box

box-shadow: 2px 2px 5px red, -2px -2px 5px blue;

  • You can add multi-color glow or a 3D effect to a single box.
  • Use rgba() for transparent shadow
  • try hover effects:

.card:hover {
            box-shadow: 0px 15px 30px rgba(0,0,0,0.4);
            transform: translateY(-5px);
            transition: all 0.3s ease;
            }
