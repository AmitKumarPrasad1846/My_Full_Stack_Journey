CSS Grid

 	CSS Grid 
    -	CSS Grid is a two-dimensional layout system used in CSS.
     It allows you to design layouts using rows and columns, 
     unlike Flexbox which is one-dimensional.
    
    	Key properties of CSS Grid:
    Property	Description
    display: grid;-----------------	Convert a container into a grid layout.
    grid-template-columns	-----------------Defines the number and size of columns.
    gird-template-rows-----------------	Defines the number and size of rows.
    gap-----------------	Sets spacing between grid items.
    grid-column	-----------------Controls and item’s column placement.
    grid-row	-----------------Controls an item’s row placement.
    fr (Fractional Unit)-----------------	Divides space in fractions (e.g., 1fr , 2fr).
    
    
    
    -------------------
    	Pro Tip:
    Use grid-template-areas for complex layouts where you want to name 
    areas and place items easily.
