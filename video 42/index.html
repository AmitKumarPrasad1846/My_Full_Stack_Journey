CSS transforms

transform property - magicbox of CSS

Syntax

transform: function(value);

scale() - Size bda/ chhota

Syntax

transform: scale(1.2); /* This will scale upto 120% f ots original size.*/

Tip:

It is perfect for hover effects.

rotate() - Ghuma do bhai

Syntax

transform: rotate(angle); /* angle = put your desired angle here. */

Element ghoom jata hai - Jese koi image ya box ko rotate kar rhe ho.

translateX() & translateY() - Idhar udhar le jao.

Syntax

transform: translateX(100px); /* 100px right */ transform: translateY(-50px); /* 50px up */

Ise apply karne par esa lagta hai jese tum element ko drag karke kisi or jagah le gaye ho.

skew() - Tedha kar do

Syntax

transform: skew(20 deg, 10deg);

Ise apply karne par lagta hai jese item thoda tilt ho gya hai - ye design me thoda drama add karta hai.

matrix() - The boss of the transform properties (Advance - nut not commonly used by the developers)

Syntax

transform: matrix(a, b, c, d, e, f);

Ye sare functions ka combo hai.

Bonus :- multiple transforms ek saath.

Syntax

transform: scale(1.2) rotate(15deg) translateX(20px);

It is like a combo , ki me gym bhi ja rha hu, dance bhi kar rh ahu or chakkar bhi kaat rha hu ek saath.
