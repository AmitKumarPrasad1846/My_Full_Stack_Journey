What are CSS Sizing units?

CSS offers a variety of sizing units to define the dimensions of elements. These units can be categorized into:

  • Absolute Units (Fixed Size)
  • Relative Units (Flexible Size)

Absolute Units (fixed size):

These units remain constant regardless of the screen size of resolution:

  • px (Pixels) = Most commonly used; 1px = 1/96th of an inch.
  • cm (Centimeters) = 1cm ≈ 37.8px.
  • mm (Millimeters) = 1mm ≈ 3.78px.
  • in (Inches) = 1in = 96px.
  • pt (Points) = 1pt = 1/72 of an inch.
  • pc (Picas) = 1pc = 12pt.

Relative Units (Flexible Size):

These units adjust based on the parent element or viewport:

  • % = Relative to the parent element.
  • em = Relative to the font-size of the element.
  • rem = Relative to the font-size of the root element (html).
  • vw = 1% of the viewport width.
  • vh = 1% of the viewport height.
  • vmin = 1% of the smaller viewport dimension.
  • vmax = 1% of the larger viewport dimension.
  • ch = Relative to the width of the "0" character.
  • ex = Relative to the x-height of the font.
