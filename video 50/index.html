What are CSS Filters?

CSS Filters allow you to apply visual effects to elements, especially images, videos, or any DOM element.
Think of them as Instagram filters... but coded.

SYNTAX:

selector {
filter: filter-function(value);
}

You can also chain Multiple filters in a single line:

selector {
filter: grayscale(100%) blur(2px) brightness(120%);
}

Common Filter Functions:

  • blur(px) → Adds a soft blur to the element. Eg: blur(4px)
  • brightness(%) → Adjusts brightness. Eg: brightness(120%)
  • contrast(%) → Controls contrast. Eg: contrast(200%)
  • grayscale(%) → Converts image to black and white. Eg: grayscale(100%)
  • sepia(%) → Gives warm brownish tint. Eg: sepia(100%)
  • invert(%) → Inverts colors like a photo negative. Eg: invert(100%)
  • hue-rotate(deg) → Rotates color wheel. Eg: hue-rotate(90deg)
  • saturate(%) → Boosts color intensity. Eg: saturate(300%)
  • drop-shadow() → Adds a styled shadow. Eg: drop-shadow(4px 4px 10px red)

Ab filters ke sath images ko style dena bacha kuch kaam nahi hai, bas filter lagao aur chamka do duniya! 🚀
