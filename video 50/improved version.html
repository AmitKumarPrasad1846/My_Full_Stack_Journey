CSS Filter Property

CSS filters allow you to apply graphical effects like blurring, color shifting, and more to elements. They're like Instagram filters but with code!

Basic Syntax

selector {
filter: filter-function(value);
}

Multiple Filters

selector {
filter: grayscale(50%) blur(2px) brightness(120%);
}

Try It Yourself

Filter Functions

  • blur(px) → Applies Gaussian blur filter: blur(5px);
  • brightness(%) → Adjusts brightness brightness(150%)
  • contrast(%) → Adjusts contrast contrast(200%)
  • drop-shadow() → Applies shadow effect drop-shadow(8px 8px 10px gray)
  • grayscale(%) → Converts to grayscale grayscale(100%)
  • hue-rotate(deg) → Shifts colors hue-rotate(90deg)
  • invert(%) → Inverts colors invert(100%)
  • opacity(%) → Sets transparency opacity(50%)
  • saturate(%) → Adjusts saturation saturate(300%)
  • sepia(%) → Applies sepia tone sepia(100%)

Pro Tip: Combine multiple filters for creative effects! 🎨
