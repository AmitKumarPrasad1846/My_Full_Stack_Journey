Object-Fit Property

The object-fit CSS property specifies how the content of a replaced element (like <img> or <video>) should be resized to fit its container.

This is particularly useful when you have fixed dimensions for your container but want to control how the media fits within those dimensions while preserving (or not) its aspect ratio.

img {
width: 300px;
height: 200px;
object-fit: cover; /* Try: fill, contain, cover, none, scale-down */
}

Object-Position Property

The object-position property determines the alignment of the content within its container. It works in conjunction with object-fit to give you precise control over how your media is displayed.

The default value is 50% 50% (centered). You can use keywords (top, left, etc.), percentages, or length values to position your content exactly where you want it.

height: 200px;
object-fit: cover;
object-position: top left; /* Try: center, 20% 80%, 50px 30px */
}

When to Use Each Value

fillStretches/squashes image to fill container (may distort): When you need the image to exactly match the container dimensions, regardless of aspect ratio.

containFits entire image within container (may leave empty space): When you want to see the entire image without cropping, even if it means empty space in the container.

coverFills container completely (may crop image): When you want the container completely filled with no empty space (most common for hero images).

noneDisplays image at original size (may overflow container): When you want the image displayed at its intrinsic size, regardless of container dimensions.

scale-downLike 'none' or 'contain' - whichever makes image smaller): When you want the image to display as if you applied either 'none' or 'contain', whichever would make the image smaller.
