CSS object-fit and object-position

What is Object-fit?

object-fit batata hai ki image ya video box ke andar kaise fit hona chahiye, bina uska aspect ratio tode hue (height/width ratio).

Ye commonly use hota hai jab image ka fixed width & height set kiya gaya ho aur fir bhi hume usse ache se fit karwana ho. Popular values: fill, contain, cover, none, scale-down

img {
width: 300px;
height: 200px;
object-fit: cover;
}
CSS

What is Object-position?

object-position define karta hai ki image/video box ke andar kis direction me placed hogi. Default value hoti hai 50% 50% (center). Isse top, left, center, percentages sab define kar sakte ho.

img {
width: 300px;
height: 200px;
object-fit: cover;
object-position: top left;
}
CSS
