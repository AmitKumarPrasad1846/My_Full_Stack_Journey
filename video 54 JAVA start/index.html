JavaScript Basics

A beginner's guide to JavaScript programming

Step 1: What is JavaScript?

JavaScript ek scripting language hai jo websites ko interactive banata hai.

  • HTML: Structure deta hai
  • CSS: Design and style deta hai
  • JavaScript: Brain & Action deta hai
Example Uses: Jab aap kisi button pe click karte hai, light/dark mode toggle hota hai, form validation hota hai, sliders chalte hai - sab JavaScript ke wajah se.
Step 2: JavaScript ko kaise use karein?

HTML ke andar JS likhne ke 3 tarike hote hain:

<!-- Method 1: Inline JS -->
<button onclick="alert('Hello Sigma')">
  Click Me
</button>
<!-- Method 2: Internal JS -->
<script>
  console.log("Hello from Internal JS!");
</script>
<!-- Method 3: External JS -->
<script src="script.js"></script>
Best Practice: External JS use karein (code alag file me likho)
Step 3: Variables in JavaScript

Variables = Containers to store data.
3 types:

var name = "Amit"; // Old style, avoid now
let age = 21; // Modern, changeable
const PI = 3.14; // Constant, fixed

Rules:

  • Variable name can't start with number
  • Can include letters, $, and _
  • Case-sensitive (Name ≠ name)
