More on CSS Selectors

  • (.element: first-child{ }) ---- It selects the first element from the parent element.
  • (* { }) ---- It is a universal selector.
  • It can also be used as a universal selector for any specific element.
    Then you can put any kind of styling inside the curly braces.
  • <style>  put your styling code inside this syntax  </style>
  • (.element::first-line{ }) ---- It will help you to select the first-line of the paragraph.
  • (.box: nth-child( ) { }) ---- It will help you to select nth element from the parent element.
  • We can also apply top, bottom, left, right - like attributes inside any block level element.
  • [data-color='primary'] ---- it help us in grouping data.
  • .element::before {content:"Amit is billionare."} ---- It wil help us in inserting content dynamically inside HTML using CSS.
  • .element::after {content:"Amit is billionare."} ---- It wil help us in inserting content dynamically inside HTML using CSS.
